# 🧪 دليل اختبار ميزات مدير الإشعارات (AetherShell)

لقد صممت هذا الملف ليكون مرجعك السريع لاختبار كافة الميزات التي قمنا بتطويرها مؤخراً لمدير الإشعارات (`osd-notify`)، ومركز الإشعارات (`panel`)، وشاشة القفل (`aetherlock`).

كل ما عليك فعله هو نسخ الأوامر التالية ولصقها في سطر الأوامر (Terminal) والتأكد من النتائج:

---

## 1. شريط التقدم (Progress Bar) 📊
اختبار ظهور شريط التقدم أسفل الإشعار بشكل أنيق (يجب أن يعمل في الإشعار المنبثق، والـ Panel، و Aetherlock).
```bash
notify-send "جاري التنزيل..." "يتم الآن تحميل تحديثات النظام" -h int:value:45
```
*غيّر الرقم `45` إلى أي رقم من `0` إلى `100` لتغيير نسبة الامتلاء.*

---

## 2. النصوص المنسقة والروابط (Markup Parser) 📝
اختبار التنسيق (العريض، المائل، الروابط). 
- في `osd-notify` و `panel`: يجب أن ترى الرابط باللون الأزرق وتستطيع النقر عليه ليفتح في المتصفح.
- في `aetherlock`: يجب أن تتجاهل الشاشة وسم الرابط دون أن تنهار، وتعرض التنسيقات الأخرى بشكل سليم.
```bash
notify-send "اختبار النصوص المنسقة" "هذا النص <b>عريض</b>، وهذا <i>مائل</i>، وهنا رابط <a href=\"https://vaxp.org\">موقع VAXP</a>."
```

---

## 3. التوقف المؤقت عند التمرير (Pause on Hover) ⏱️
اختبار إيقاف مؤقت الإشعار عند وضع مؤشر الفأرة (Mouse) فوقه.
- الأمر التالي سيرسل إشعاراً يختفي بعد 3 ثوانٍ فقط (`-t 3000`).
- قم بتحريك الفأرة فوراً فوق الإشعار، **يجب ألا يختفي** طالما أن الفأرة فوقه.
```bash
notify-send "توقف مؤقت" "هذا الإشعار يختفي بسرعة. ضع الفأرة فوقي لإيقاف العداد!" -t 3000
```

---

## 4. إشعارات التحذير (Critical Notifications) ⚠️
اختبار الصوت المخصص للتحذيرات، وعدم اختفاء الإشعار بسرعة.
```bash
notify-send -u critical "تحذير أمني" "محاولة تسجيل دخول فاشلة غير مصرح بها."
```

---

## 5. اختبار أيقونات النظام (System Icons) 🖼️
اختبار قدرة الإشعار على جلب وعرض الأيقونات من النظام مباشرة بدلاً من أيقونة الافتراضية.
```bash
notify-send "إعدادات الشبكة" "تم الاتصال بشبكة Wi-Fi بنجاح" -i network-wireless
```
أو أيقونة أخرى:
```bash
notify-send "البطارية منخفضة" "يرجى توصيل الشاحن فوراً." -i battery-empty
```

---

### ✅ نصيحة أخيرة للمراجعة (Verification)
بعد تشغيل هذه الأوامر جميعاً:
1. انقر على أيقونة الإشعارات في **الشريط السفلي (Panel)** وتأكد أن جميع الإشعارات السابقة مسجلة بشكل جميل وصحيح.
2. قم بقفل الشاشة (Lock Screen) لتتأكد من أن **Aetherlock** يعرض شريط التقدم، والنصوص العريضة، وأنه لم يتعطل.

إذا عملت جميع هذه الأوامر بشكل سليم، فهذا يعني أن التحديث جاهز 100%، ويمكننا التوجه إلى الإضافة القادمة (مثل خاصية الرد المباشر Inline Reply)! 🚀
